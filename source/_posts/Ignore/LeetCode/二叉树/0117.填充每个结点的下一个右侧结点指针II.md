## 方法一：层序遍历

利用层序遍历，即可找到每一层的从左到右的结点，使用一个额外指针记录上一个结点即可

```cpp
Node* connect(Node* root) {
    queue<Node*> que;
    if(root) que.push(root);
    while(!que.empty())
    {
        int size = que.size();
        Node* cur = nullptr;
        Node* pre = nullptr;
        for(int i=0;i<size;i++)
        {
            cur = que.front(); que.pop();
            if(i!=0)
                pre->next = cur;
            if(cur->left) que.push(cur->left);
            if(cur->right) que.push(cur->right);
            pre = cur;
        }
        cur->next = nullptr;
    }
    return root;
}
```
